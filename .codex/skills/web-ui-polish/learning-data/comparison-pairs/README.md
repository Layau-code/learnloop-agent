# Comparison Pairs

This directory contains before/after comparison images showing AetherPane improvements.

## Naming Convention
`before-after-[page-type].png`

Examples:
- `before-after-landing.png`
- `before-after-dashboard.png`
- `before-after-showcase.png`

## Format
Each image should be a side-by-side comparison:
- Left side: Before (original rough page)
- Right side: After (polished AetherPane page)
- Divider line or label separating the two versions

## How to Add
1. Capture the before state of a page
2. Apply AetherPane polish
3. Capture the after state
4. Create a side-by-side comparison image (1920x540 recommended per side)
5. Name it according to convention
6. Place it in this directory
