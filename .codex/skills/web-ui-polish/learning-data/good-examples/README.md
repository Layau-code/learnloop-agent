# Good Examples

This directory contains screenshots of high-quality UI implementations that follow AetherPane design principles.

## Naming Convention
`[page-type]-[theme]-[aspect].png`

Examples:
- `landing-classic-hero.png`
- `dashboard-signal-metrics.png`
- `showcase-editorial-grid.png`

## Quality Criteria
- Visual Hierarchy Score: 8+
- Breathing Space Score: 8+
- Glass Quality Score: 8+
- Overall Design Score: 8+

## How to Add
1. Generate or find a high-quality example
2. Take a screenshot (1920x1080 recommended)
3. Name it according to convention
4. Place it in this directory
